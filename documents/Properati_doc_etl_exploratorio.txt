Documentación del Proceso de Extracción y Análisis Exploratorio en Properati
Este documento resume el proceso completo seguido para resolver los problemas de extracción y exploración de datos de propiedades del portal Properati, abordando tanto los desafíos iniciales de extracción como el análisis exploratorio de los datos obtenidos.

1. Extracción de Datos desde HTML (Properati)
Problema Inicial:
Los datos de Properati se encontraban en formato HTML, y necesitábamos extraer información clave como precio, ubicación, área, habitaciones, baños, agencia y fecha de publicación. Un reto particular fue la correcta identificación y extracción de los valores numéricos (especialmente el área, precio, habitaciones y baños) y la exclusión de propiedades que estaban etiquetadas como "PROYECTO".

Solución Implementada:
Para resolver esto, implementamos un script en Python que utiliza la librería BeautifulSoup para extraer y limpiar los datos relevantes. A continuación, se detalla el flujo general:

Carga del archivo HTML:

Se definió una función que recibe un archivo HTML, lo procesa y busca las etiquetas de información relevante utilizando BeautifulSoup.
Extracción de campos clave:

Se definieron funciones específicas para extraer campos como el precio, la ubicación, el área, el número de habitaciones, los baños y la agencia.
El precio y el área se limpiaron utilizando expresiones regulares para asegurarnos de que solo se extrajeran los valores numéricos.
Se implementó un filtro para excluir propiedades etiquetadas como "PROYECTO", de modo que solo se extrajeran las propiedades con la etiqueta "N/A".
Limpieza de Texto:

Cada campo extraído fue limpiado de caracteres innecesarios como espacios adicionales o etiquetas HTML no deseadas.
Exportación a CSV:

Una vez extraídos los datos, se exportaron a archivos CSV con un formato separado por | para facilitar el análisis posterior.
Procesamiento de múltiples archivos:

El script fue configurado para procesar múltiples archivos HTML, identificando aquellos que correspondían a Properati y generando un archivo CSV por cada uno.

2. Análisis Exploratorio de los Datos de Properati
Una vez extraídos los datos, realizamos un análisis exploratorio con el objetivo de entender las tendencias del mercado inmobiliario en Quito. Este análisis incluyó la visualización de la distribución de precios, áreas, número de habitaciones, y la comparación entre barrios y agencias.

Problemas Identificados Durante el Análisis:
Algunas columnas, como "Fecha de Publicación", no proporcionaron información útil en el análisis, por lo que decidimos omitirlas.
La correlación entre el área y el precio fue una de las relaciones más significativas, mostrando una fuerte correlación positiva.
Hubo problemas menores con la representación gráfica debido a advertencias de versiones futuras en seaborn, que fueron corregidas para evitar futuros errores.
Análisis Exploratorio Realizado:
Distribución de Precios: Mostramos la distribución de precios, que reveló una concentración en el rango de $100,000 a $200,000.

Relación entre Precio y Área: Se observó una relación clara entre el tamaño de la propiedad y su precio, con propiedades más grandes tendiendo a precios más altos.

Distribución de Áreas: Mostramos la distribución de áreas de las propiedades, revelando que la mayoría de las propiedades están entre 75 m² y 150 m².

Distribución de Precios por Habitaciones: Se visualizó cómo el número de habitaciones afecta el precio, destacando que las propiedades de 3 habitaciones mostraban una mayor variabilidad en los precios.

Distribución del Número de Baños: Se observó que la mayoría de las propiedades tenían 2 baños, mientras que propiedades con más de 3 baños eran raras.

Distribución de Precios por Barrio: Analizamos los precios por diferentes barrios, con zonas como Cumbayá y Centro Norte destacando por su elevado precio promedio.

Distribución de Precios por Agencia: Se visualizó la dispersión de precios según la agencia inmobiliaria, lo que mostró la especialización de algunas agencias en propiedades de lujo.

Heatmap de Correlaciones: Se realizó un análisis de correlación entre las variables numéricas (precio, área, habitaciones y baños), mostrando una fuerte correlación entre área y precio.

Relación entre Área y Número de Habitaciones: Se exploró cómo el tamaño de las propiedades se relaciona con el número de habitaciones.

Resultados Finales:
El análisis reveló que el mercado de propiedades en Quito está claramente orientado hacia el rango medio de precios, con una fuerte demanda en propiedades de entre $100,000 y $200,000 y de tamaño medio (75 m² - 150 m²). Se observó una fuerte correlación entre el área y el precio, aunque algunas propiedades más pequeñas lograron precios elevados, probablemente debido a factores externos como la ubicación o las características adicionales.

3. Conclusiones y Recomendaciones Finales
El mercado de propiedades en Quito se encuentra principalmente en un segmento medio, lo que refleja un interés masivo en propiedades accesibles para familias.
Existen oportunidades en el mercado de lujo, aunque es limitado. Los barrios más caros, como Cumbayá y Centro Norte, tienen una amplia gama de propiedades que se mueven en un rango de precios mucho más amplio.
El tamaño de la propiedad es el factor más influyente en el precio, pero es necesario tener en cuenta otros factores, como la ubicación y las amenidades, que también impactan considerablemente.
Este proceso integral de extracción y análisis proporciona una visión completa del mercado inmobiliario de Quito y ofrece herramientas valiosas para los desarrolladores, inversores y compradores que buscan tomar decisiones basadas en datos.
